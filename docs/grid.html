    path_coordinates() is a helpful enumeration of the eight possible paths as arrays of coordinate objects. Iterating over paths with paths(), the current index of the iteration can be passed to this path_coordinates() result to find the actual coordinates of possible moves.

                path_coordinates: function() {
                return [
                    /* rows */
                    [{x: 0, y: 0}, {x: 1, y: 0}, {x: 2, y: 0}],
